我的创业

发表于 2019-03-02 | 分类于 杂谈

每每被面试，必定会被问到当初为何创业？如果说想趁着年轻，就想创业，不想让自己未来后悔，仿佛并不能令人信服。

我是这么看待的：从两个角度来看，企业和我自己。

从企业角度，更看重你入职后是否稳定，别工作一段时间，又去辞职创业了，毕竟招人也是有成本的。另外，如果创业不是继续做技术写代码，其实从毕业到现在的所谓工作经验，要减去创业时间的，这样虽然号称6年工作经验，实际上只有4年多。企业招人更多的是看重性价比，有的企业说喜欢创业的员工，其实隐藏的意思：在实力基础上，创业是锦上添花，大多数情况都是减分的。

我这人比较实诚，从来不在简历上作假，这是做人的底线，不会为了规避自己的劣势而刻意修改简历，是这样就是这样，不违心，生活的坦荡。

从我自身来看，现在回想起来，虽然少挣了一年多的钱，后面过的艰难些，让家人也受累了，自己需要更加努力才能赶上同龄的人，却从来不后悔。自己选择的路，咬着牙也要走下去。当然自己的收获更多：

  • 经历过焦虑和压力，后面更知道自己想要什么，自己在哪些方面缺失了，需要补习哪些知识，做事情更有目的
  • 从创业的坑里爬出来后，对风险的认识更上一层了，也逼迫自己学习识别风险，评估风险，增加自己的抵抗能力
  • 见过世面了，不仅仅局限在技术的小世界中，知道营销、商业、运营等等都是自己一生中重要的技能，而不仅仅是编程。有句话：手里拿着锤子，觉得到处都是钉子
  • 在创业中，遇到形形色色的人，颠覆了从校园学到的认知。刚毕业那会，觉得这世界的人，都是跟校园的同学一样，沟通和素质都跟自己类似，也都是good guys，其实不是，他们既不是好人也不是坏人，不同的情况，表现不一样，仅此而已。曾经，我总是把别人先当做好人来看，别人对我好一分，我要对别人好十分。后来吧，我发现不是那么回事，真是too young, too simple
  • 创业更多的是一种态度，一种随时都在创业的态度。如果非要开公司，拉几个人，融点资金才叫创业，那就太狭隘了。如果创业为了暴富，往往会欲速则不达，适得其反，100%会失败，因为格局不够。
  • 一定要建立个人品牌，这个认识是误打误撞的，本来写公众号是为了营销卖儿童编程课程，却无意中认识不少人，也挺逗的

暂时只能想到这些，这些话可能面试中不会说，却是真实的。如果过去的经历对自己的成长没有一点帮助，那真是白经历了，浪费了时间不说，还增加了沉没成本。改天想到些什么，再添加吧！
